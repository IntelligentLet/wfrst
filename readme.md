# WFRST
A better name would be cool...

Install the following:
```
alacritty
cava
i3-gaps
neovim
picom
polybar
rofi
wal
wallpapers
```

To install, run

`cp -r config/* /home/$USER/.config/`

`cp -r .oh-my-zsh /home/$USER/`

`cp .zshrc /home/$USER/`
